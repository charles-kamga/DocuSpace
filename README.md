# DocuSpace
Docuspace est un espace de travail en ligne qui permet de stocker et de gérer des documents.

## Fonctionnalités
- Création et gestion de dossiers
- Upload et gestion de documents
- Suppression sécurisée des documents et dossiers
- Interface utilisateur intuitive

## Technologies utilisées
- Python 3.12
- Django 5.2.8
- Bootstrap 5.3.2
- jQuery 3.7.1
